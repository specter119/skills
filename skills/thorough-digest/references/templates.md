## Quick Start Example

```markdown
User: "Help me organize 15 research notes under research/ and extract key findings"

→ Phase 1: Analyze

  [Inventory]
  Found 15 markdown files in research/

  [Classify]
  - Group A: notes-01 to notes-05
  - Group B: notes-06 to notes-10
  - Group C: notes-11 to notes-15

  [Plan]
  "Starting to process 15 research notes

   Output directory: ./thorough-digest/notes-summary/

   Groups:
   - Group A: 5 items (notes-01 to 05)
   - Group B: 5 items (notes-06 to 10)
   - Group C: 5 items (notes-11 to 15)

   Launching 3 parallel tasks..."

→ Phase 2: Parallel Process

  - Sub-agent A processes notes 1-5 → group-a/findings.md
  - Sub-agent B processes notes 6-10 → group-b/findings.md
  - Sub-agent C processes notes 11-15 → group-c/findings.md

→ Phase 3: Synthesize

  [Collect]
  All 3 groups completed, 15/15 items processed

  [Gap Analysis]
  - Item 7 mentions "A2A protocol" but no details
  → Trigger Deep Research for A2A background

  [Synthesize]
  Combine all findings + Deep Research results
  → Output: synthesis.md

→ Done

  "Processing complete

   Report: ./thorough-digest/notes-summary/synthesis.md
   Processed: 15 notes in 3 parallel groups

   Key findings:
   1. [Theme 1]
   2. [Theme 2]
   3. [Theme 3]

   Supplemented: A2A protocol background (via Deep Research)"
```

## Gap Analysis

```markdown
## Gap Analysis

### Coverage Check
- [ ] All items processed?
- [ ] All required fields extracted?
- [ ] Any items with incomplete data?

### Information Gaps
1. [Gap 1]: Missing [what] for items [which]
2. [Gap 2]: Need external info about [topic]

### Resolution
- Gap 1: Re-process specific items
- Gap 2: **Trigger Deep Research** for external information
```
